# SNV_recurrence_analysis_tool
This is a recurrence analysis tool for SNP/Indel analysis.
Due to the unique vcf formats among different platforms, this analysis is TOOL SPECIFIC currently.
 
Test01 ,Test2, Test3.txt were extract from ANNOVAR annotation files of DRAGEN WGS somatic pipeline results.
union_of_test_variants.txt were made from those ANNOVAR.txt files with "find_union.sh" on linux.
